Das Package dient als Ergaenzung zum Leitprogramm 3D-Graphikprogrammierung.

Es bietet unter anderem die folgende Funktionalitaet:

  • Klassen zur Verwaltung der Eigenschaften (Ecken, Kanten und Flaechen) von Polyedern.
  • Einlesen von Daten aus ASCII-Files im Wavefront Object Format.
  • Beispielprogramm als Applet und Applikation zu Demonstrationszwecken.
